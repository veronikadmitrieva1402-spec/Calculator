# Calculator

Дмитриева Вероника ЭФБО-04-24

## Функционал

Программа включает в себя простой калькулятор в консольном приложении на языке C#. Пользователь поэтапно вводит числа, применяя возможные бинарные (+, -, *, /, %) и унарные (1/x, x^2, sqrt) операции. Работа с памятью производится с помощью M+, M-, MR. M+ - добавление к памяти. M- - вычитание из памяти, MR - вызов из памяти.
