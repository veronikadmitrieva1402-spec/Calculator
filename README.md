## Функционал

Программа включает в себя простой калькулятор в консольном приложении на языке C#. Пользователь поэтапно вводит числа, применяя возможные бинарные (+, -, *, /, %) и унарные (1/x, x^2, sqrt) операции. Работа с памятью производится с помощью M+, M-, MR. M+ - добавление к памяти. M- - вычитание из памяти, MR - вызов из памяти.

## Ограничения

Операции следуют строго поэтапно.

Нет возможности проводить операции с особо крупными числами.

Все операции производятся к текущему значению. В случае чего есть возможность очистки памяти. Нельзя провети отмену операции.

## Ошибки

При некорректных значениях (буквы, знаки) пользователь получает ошибку и программа закрывается.

При делении на 0, программа выдает ошибку "Ошибка! На 0 делить строго запрещено!"

Получение корня из отрицательного числа невозможно "Ошибка! Нельзя извлечь корень из отрицательного числа!"
